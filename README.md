# 前后端分离个人健康管理系统 java227

## 项目概述

本项目是一款基于前后端分离架构的个人健康管理系统，旨在为用户提供全面的健康数据管理及可视化分析功能。

## 技术栈

- 后端：Spring Boot + MyBatis-Plus
- 前端：Vue2 + ElementUI + ECharts + Vuex
- 数据库：MySQL、Redis

## 功能模块

### 用户管理

- 注册与登录
- 系统可视化统计信息首页

### 系统角色

- 普通用户
- 超级管理员

### 用户操作

- 用户管理：增删改查
- 角色管理

### 运动知识管理

- 新增、修改、删除运动知识
- 搜索运动知识

### 运动详情管理

- 新增、修改、删除运动详情
- 搜索运动详情

### 身体信息管理

- 修改、删除用户身体信息
- 搜索用户身体信息
- 身体信息上传

### 健康评估

- 通过ECharts进行健康信息可视化统计

### 健康信息管理

- 编辑和删除健康信息

## 系统角色

- 普通用户：可管理个人信息、上传身体数据、查看健康评估等。
- 超级管理员：负责系统用户管理、角色分配及运动知识的管理。

## 运行环境

- 后端：Java 1.8+，Spring Boot 2.x，MyBatis-Plus 3.x
- 前端：Vue 2.x，Node.js 12.x，ElementUI 2.x，ECharts 5.x
- 数据库：MySQL 5.7+，Redis 3.2+

## 目录结构

```
项目根目录/
│
├── backend/          # 后端代码目录
│   ├── src/
│   └── pom.xml
│
└── frontend/         # 前端代码目录
    ├── src/
    ├── public/
    └── package.json
```

## 部署步骤

1. 准备好环境：安装Java、Node.js、MySQL和Redis。
2. 分别进入`backend`和`frontend`目录，执行对应的构建和部署脚本。
3. 部署前端静态文件到Web服务器。
4. 部署后端服务到应用服务器。

## 核心亮点

- 基于前后端分离模式，提供高效、稳定的服务。
- 使用Vue和ElementUI构建用户友好的界面。
- 结合ECharts进行健康数据可视化展示，直观了解健康状态。
- 使用Spring Boot和MyBatis-Plus，确保后端的高效和数据安全。
